Ti sei mai trovato in questa situazione: stai utilizzando un'app (come un'app bancaria) e devi esportare urgentemente un file (ad esempio, un'estratto conto in PDF). Vuoi scaricare quel file localmente sul tuo dispositivo, ma l'app non offre un modo semplice per farlo; il massimo che offre è la possibilità di condividere il file, il che significa che finisci per doverlo inviare a te stesso (o a qualcun'altro) tramite e-mail o un'app di messaggistica istantanea, per poi avere la possibilità di salvarlo solo a quel punto.

Questo accade perché molte app scelgono di utilizzare la funzionalità di condivisione del sistema invece di implementare la logica di salvataggio dei file richiesta. Quindi, cosa si può fare?

In questi casi, " Save Locally" potrebbe tornare utile: basta selezionarla nella finestra di condivisione che si apre e ti consente di salvare il file ovunque localmente sul tuo dispositivo, mostrando anche i dettagli del file in anticipo (anche se questo può essere saltato). È disponibile anche un'opzione per intercettare gli intenti "Apri con", consentendo di salvare i file anche se l'app sorgente tentasse di forzare la visualizzazione con un visualizzatore predefinito per quel tipo di file.

Save Locally è un software FLOSS (Free and Libre Open Source Software) sviluppato con Jetpack Compose. Il codice sorgente è disponibile su <a href="https://github.com/MateusRodCosta/SaveLocally">https://github.com/MateusRodCosta/SaveLocally</a>, sotto licenza AGPLv3+.
